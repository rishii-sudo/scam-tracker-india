# 🇮🇳 Scam Tracker India

A real-time scam tracking platform for India with an interactive map.

![Scam Tracker India](https://scam-tracker-india.vercel.app)

## 🌐 Live Demo
👉 [scam-tracker-india.vercel.app](https://scam-tracker-india.vercel.app)

## 📌 Features
- 🗺️ Interactive dark map with scam locations across India
- 🔴 Color-coded dots by scam type
- 🔍 Filter scams by category
- 💸 Total amount lost counter
- 🚨 Report a scam form — data saves to database instantly
- 📊 Real-time data from Supabase

## 🛠️ Tech Stack
- **Frontend:** React.js + Tailwind CSS
- **Map:** Leaflet.js
- **Database:** Supabase (PostgreSQL)
- **Deploy:** Vercel

## 📂 Data Sources
- CERT-In (Indian Computer Emergency Response Team)
- RBI Fraud Alerts
- User Reports

## 🚀 Run Locally
```bash
git clone https://github.com/rishii-sudo/scam-tracker-india.git
cd scam-tracker-india
npm install
npm start
```
